แนวคิด

จุดเด่น

แอปพลิเคชันนี้ีจุดเด่นอยู่ที่ต้องใช้ Information Retrieval วิเคราะห์ข้อมูลก่อนจึงจะสามารถนำมาใช้ได้

เทคโนโลยีที่ใช้

แอปพลิเคชันนี้พัฒนาโดยใช้ Ionic Framework และเว็บเซอร์วิซใช้ Codeigniter ในการพัฒนา

เวลาที่ใช้พัฒนา

แอปพลิเคชันนี้ใช้ระยะเวลา 3 เดือนในการพัฒนาโดยมีความยากอยู่ที่การพัฒนาในส่วนของเว็บเซอร์วิซ

E-commerce

หากคุณสนใจสามารถดูตัวอย่างเต็มๆได้ ทีนี่

แนวคิด

เป็นเว็บไซด์ e-commerce พร้อมระบบจัดการเว็บไซด์โดยพัฒนาขึ้นตามความความต้องการลักษณะธุรกิจของผู้ว่าจ้าง

จุดเด่น

พัฒนาขึ้นตามรูปแบบที่ผู้ใช้ต้องการทำให้เว็บไซด์มีเอกลักษณ์เฉพาะตัวตามรูปแบบธุรกิจของผู้ใช้รวมถึงมีการจัดการภายในทำให้เว็บไซด์สามารถดูแลรักษาได้ง่าย

เทคโนโลยีที่ใช้

เว็บไซด์นี้ใช้ PHP ในการพัฒนาและมี MySQL เป็นฐานข้อมูลนอกจากยังนี้ใช้ Bootstrap และ jQeury ร่วมในการพัฒนาด้วย

เวลาที่ใช้พัฒนา

เว็บไซด์นี้ใช้เวลาในการพัฒนาประมาณ 1 เดือนโดยส่วนที่ยากในการพัฒนาคือการวิเคราะห์ความต้องการของผู้ใช้เพื่อพัฒนาเว็บไซด์ให้ถูกต้องตามความต้องการ
