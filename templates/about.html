เกี่ยวกับเรา

เรื่องทุกอย่างเกี่ยวกับ Noob Dev ที่คุณอยากรู้ และไม่อยากรู้

สวัสดีครับผมชื่อปฐมพร เมืองคำอายุ 22 ปีจบมาจากมหาวิทยาลัยชื่อดังย่านบางแสน คณะวิทยาการสารสนเทศ สาขาวิศวกรรมซอฟแวร์ มีความชื่นชอบในการเขียนโปรแกรมและหลงรัก Open Source ถึงขนาดใช้ Linux เป็น OS หลักจนแทบเรียนไม่จบ (มีปัญหากับการทำรายงาน = =" ) งานอดิเรกในเวลาว่างนอกจากเล่นเกมก็ขียนโปรแกรมต่างๆเล่นมีประสบการณ์เป็น Freelance มา 2 ปีแล้ว สามารถเขียนโปรแกรมภาษาต่างๆที่สามารถเขียนได้ฟรีเช่น ภาษาพื้นฐานอย่าง PHP และ Framework ของ PHP อย่าง Codeigniter (โดนบังคับให้เขียนตั้งแต่ปี 2) JAVA (สำหรับพัฒนาแอปพลิเคชันบน Android) แต่สำหรับภาษาที่ผมหลงรักที่สุดคือ Javascript ครับเรียกว่าหลงรักอย่างสุดหัวใจไม่ว่าจะเป็น jQuery, AngularJS, Node.js, MongoDB, Ionic Framework และเทรนที่กำลังมาแรงในขณะนี้อย่าง MEAN Stack สุดท้ายนี้ต้องให้เครดิตแก่อาจารย์ของผมที่สาขาวิศวกรรมซอฟแวร์ที่ช่วยสอนพื้นฐานต่างๆ และอาจารย์ที่เป็น Idol ของผมแต่เค้าไม่รู้อิอิ Devahoy หวังว่าจะได้รับใช้ทุกท่านนะครับขอบคุณครับ
